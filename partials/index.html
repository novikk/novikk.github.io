What is URB?

URB is a new game mode for League of Legends. It consists on giving both teams random champs and builds. You then download the itemset for your build and head right into the game. This mode is meant to be played in the ARAM map.

What are item sets?

An item set is a custom set of items that are displayed within the in-game item shop. They are included in a drop down along with the Recommended Items. An example item set for an URB game is shown in the image
